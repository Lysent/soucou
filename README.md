# Soucou
Story Bullet Hell Game

[Game]
## Gameplay
Dodge or corrupt bullets to survive. Corrupt enemies at close range to progress.

**Controls**<br>
WASD / ↑←↓→ - Movement<br>
C / X - Corruption (Attack)<br>
Hold C / Hold X / - Special Corruption (Attack)<br>
N / Esc - Pause
